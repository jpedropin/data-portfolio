# Data-Portfolio
1) Projeto completo de web-scrapping usando (Selenium + BeautifulSoup + Postgres em docker)

Coleta título, categoria, estrelas, preço e estoque de http://books.toscrape.com/
Navegação com Selenium, parse com BeautifulSoup, hospedado em Postgres com uso de Docker.

Link para o projeto: https://github.com/jpedropin/books-scraper


2) Projeto completo de ETL usando técnicas de anonimização de dados, baseada na LGPD e na HIPAA (Pandas + GCP + Python + LGPD e HIPAA)

Desenvolvimento de pipeline ETL end-to-end na GCP (BigQuery). Implementação de algoritmos de Hashing com Salt em Python para anonimização de dados sensíveis em conformidade com LGPD e HIPAA.

Link para o projeto: https://github.com/jpedropin/data-encryption
